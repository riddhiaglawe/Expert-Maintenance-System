# 🧠 How It Works  

The system uses a **Rule-Based Expert System** approach:
Input Data → Knowledge Base → Inference Engine → Output Decision

## Knowledge Base (Example Rules)

- IF Temperature > 85 AND Vibration = High → Bearing Wear  
- IF Oil Level = Low → Refill Lubrication  
- IF Battery Status = Low → Replace Battery  
- IF Vibration = High AND Noise = High → Shaft Misalignment  
- IF Temperature > 90 → Critical Overheating  

## AI Processing

- Evaluates input parameters  
- Applies rules using inference engine  
- Generates maintenance suggestions  
- Displays reasoning (Explainable AI)

# 🔄 State Space Search  

The system models decision-making as a **state-space search process**:
Initial Motor State] → [Temperature > 85] → [Oil Level Low] → [Fault Detected]

- Each block = State
